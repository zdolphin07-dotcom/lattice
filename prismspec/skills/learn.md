# Skill: learn — PrismSpec Learning

**Triggers**: `/learn`, capture lesson, remember, durable knowledge

## Capability

Capture reusable lessons discovered during a PrismSpec run.

## Workflow

1. Keep only durable, future-useful knowledge.
2. Do not store secrets, raw user data, or one-off implementation trivia.
3. If Lattice exists, write to `lattice/knowledge/` and update `lattice/knowledge/index.md`.
4. Otherwise write to `prismspec/knowledge/` and maintain a short `index.md`.

## Exit Criteria

- The lesson is discoverable later.
- The note includes scope, trigger context, and practical guidance.
