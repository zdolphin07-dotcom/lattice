Create or update PrismSpec context and spec artifacts.

Execute `prismspec/skills/prismspec-specification/SKILL.md`.

Run `bash prismspec/bin/guide.sh $ARGUMENTS --from=specification --json`, then follow the returned skill.

If the requirement is too ambiguous to write a formal spec, enter PrismSpec grilling mode first: execute `prismspec/skills/prismspec-grilling/SKILL.md`, ask one engineering-boundary question at a time with a recommended answer, and keep the draft at `status: clarifying` until scope, ACs, execution mode, verification plan, and approval are concrete.

User input: $ARGUMENTS
